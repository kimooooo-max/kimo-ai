# 카드 기장 도우미

카드 사용 내역 캡쳐 이미지를 첨부하면 OCR로 날짜, 금액, 사용처, 카테고리를 추출하는 PWA입니다.

## iPhone 단독 사용

이 앱은 GitHub Pages 같은 HTTPS 정적 호스팅에 올린 뒤 iPhone Safari에서 `홈 화면에 추가`해서 사용할 수 있습니다.

브라우저 단독 모드에서는 `Tesseract.js`가 iPhone 안에서 OCR을 실행합니다. 첫 실행에는 OCR 라이브러리와 언어 데이터를 내려받기 때문에 인터넷 연결이 필요합니다.

## 주요 기능

- 카드 승인 문자 캡쳐 OCR
- 여러 승인 문자 자동 분리
- 실제 사용 금액 추출
- 누적 금액 제외
- 날짜, 사용처, 카테고리 자동 입력
- 로컬 저장 및 CSV 내보내기
- iPhone 홈화면 PWA 지원

## Mac 로컬 OCR 모드

Mac에서 더 빠르고 정확한 OCR을 쓰려면 아래 명령으로 로컬 서버를 실행합니다.

```bash
python3 server.py
```

서버가 켜진 상태에서 `http://127.0.0.1:4173` 또는 같은 Wi-Fi의 iPhone 접속 주소로 열면 macOS Vision OCR을 우선 사용합니다.

## 보너스: 자취 저녁 룰렛 🎲

저녁 한 끼 뭘 먹을지 고민이 될 때 쓰는 랜덤 메뉴 추천기입니다. `dinner/` 폴더에 있는 독립 페이지이며, 같은 GitHub Pages로 함께 배포됩니다.

- 버튼 한 번이면 룰렛이 돌고 오늘 저녁 메뉴가 정해집니다.
- 자취생 1인분 기준으로 현실적으로 해먹을 수 있는 메뉴 72종 수록
- **종류**(밥·면·국물·간단·든든)와 **난이도**(초간단·보통·제대로)로 필터
- 메뉴마다 예상 조리 시간, 재료, 한 줄 꿀팁 제공
- 유튜브·만개의레시피 레시피 바로가기
- 마음에 드는 메뉴 찜(즐겨찾기, 로컬 저장)
- 오프라인에서도 동작하는 PWA — iPhone 홈 화면에 따로 추가 가능

배포 후 주소:

```text
https://kimooooo-max.github.io/kimo-ai/dinner/
```

로컬에서 바로 보려면 저장소 폴더에서 아래처럼 정적 서버를 띄우고 `/dinner/` 로 접속하면 됩니다.

```bash
python3 -m http.server 4173
# http://127.0.0.1:4173/dinner/
```
