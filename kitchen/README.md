# 자취 부엌 🍳

저녁 메뉴 추천 + 우리집 냉장고 재료 관리 + 냉장고 재료로 만드는 요리 추천 + 밀프랩 추천을 한 파일에 담은 **단독 웹앱**입니다. 빌드·서버·설치가 필요 없고 `index.html` 하나로 동작합니다.

## 기능

- **🎲 추천** — 종류 · 난이도 · **인원(🧍1인 / 👥2인 / 👨‍👩‍👧‍👦대용량 4인+)** 필터로 랜덤 메뉴 추천 (메뉴 121종)
- **🧊 냉장고** — 가진 재료를 아이콘으로 넣고 빼며 관리
- **🍳 냉장고요리** — 냉장고에 있는 재료로 만들 수 있는 메뉴 추천 (바로 가능 / 거의 가능)
- **🥡 밀프랩** — 한 번에 만들어 두고 여러 끼 먹는 밀프랩 메뉴 (분량 · 보관 기간 · 꿀팁)
- **📝 메뉴** — 메뉴 직접 추가 / 수정 / 삭제 (인원도 지정 가능)

찜 · 냉장고 재료 · 직접 추가한 메뉴는 브라우저(localStorage)에 저장됩니다.

## 로컬에서 바로 쓰기

`index.html`을 더블클릭해 브라우저로 열면 끝입니다. (인터넷은 레시피 링크를 누를 때만 필요)

## 웹에 배포 (GitHub Pages)

이 폴더는 같은 저장소의 GitHub Pages로 함께 배포됩니다.

- 배포 주소: <https://kimooooo-max.github.io/kimo-ai/kitchen/>
- `main` 브랜치에 반영되면 자동 빌드됩니다 (`.github/workflows/pages.yml`).
- 최초 1회만: 저장소 **Settings → Pages → Build and deployment → GitHub Actions** 로 설정.

## 아이폰에 앱처럼 설치 (PWA)

전용 아이콘 · 앱 이름 · 오프라인 실행을 지원합니다.

1. **Safari**로 배포 주소 접속 (크롬 말고 꼭 Safari)
2. 하단 **공유 버튼** → **홈 화면에 추가**
3. 자동으로 **요리사 캐릭터 아이콘 + "자취부엌"** 이름이 잡힘 → **추가**
4. 홈 화면 아이콘으로 실행하면 주소창 없는 전체화면 앱처럼 열림

> 한 번 열어두면 서비스워커가 앱을 캐시해 그 다음부터는 **인터넷 없이도** 실행됩니다. (레시피 바로가기 링크를 누를 때만 인터넷 필요)

### 구성 파일

- `index.html` — 앱 본체(단일 파일)
- `manifest.webmanifest` — 앱 이름 · 아이콘 · 전체화면 설정
- `sw.js` — 오프라인 캐시(서비스워커)
- `icon-180.png` / `icon-192.png` / `icon-512.png` — 홈 화면 · 스플래시 아이콘
