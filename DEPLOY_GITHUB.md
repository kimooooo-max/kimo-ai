# GitHub Pages 배포

이 저장소에는 카드 기장 도우미 PWA 파일과 GitHub Pages Actions 워크플로가 포함되어 있습니다.

## 현재 필요한 GitHub 설정

저장소가 private이면 iPhone에서 외부 공개 URL로 접속하지 못할 수 있습니다. iPhone 외부 사용 목적이면 저장소를 public으로 전환하거나, GitHub Pages가 private Pages를 지원하는 플랜/설정인지 확인해야 합니다.

## Pages 활성화

GitHub 웹에서 아래 경로로 이동합니다.

1. Repository `kimooooo-max/kimo-ai`
2. `Settings`
3. `Pages`
4. `Build and deployment`를 `GitHub Actions`로 설정
5. 워크플로 실행 완료 후 아래 주소 확인

```text
https://kimooooo-max.github.io/kimo-ai/
```

## iPhone 홈 화면 추가

1. iPhone Safari에서 Pages 주소 접속
2. 공유 버튼
3. `홈 화면에 추가`
4. `카드기장` 앱 실행
